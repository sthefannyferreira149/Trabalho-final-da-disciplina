*NewsHub - Portal de Notícias*

*Sobre o Projeto*

O NewsHub é um sistema web desenvolvido em PHP e SQLite que permite o cadastro e autenticação de usuários para acesso a um portal de notícias.

O objetivo do projeto é oferecer uma experiência segura de login e acesso restrito a conteúdos informativos, organizados em diferentes categorias de interesse.

Para acessar as notícias, o usuário deve criar uma conta, realizar o login e, após autenticado, poderá navegar pelo portal. Ao encerrar a sessão, o sistema redireciona o usuário para a tela de login, garantindo maior segurança no acesso às informações.

*Funcionalidades:*

Cadastro de Usuário
Criação de novas contas.
Validação de dados informados.
Armazenamento seguro das informações no banco SQLite.

*Login Seguro:*

Autenticação de usuários cadastrados.
Controle de sessão.
Restrição de acesso para usuários não autenticados.

*Portal de Notícias:*

Após o login, o usuário pode acessar notícias organizadas nas seguintes categorias:

Futebol,
Fórmula 1,
Viagens,
Gastronomia e Comida.

*Logout:*

Encerramento seguro da sessão.
Retorno automático para a tela de login.

*Tecnologias Utilizadas:*

PHP,
SQLite,
PDO,
HTML,
CSS e
JavaScript.
